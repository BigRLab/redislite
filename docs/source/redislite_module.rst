
Extended Redis bindings with  embedded redis server classes
***********************************************************

The :mod:`redislite` module contains a redis-server and extended versions of the :class:`redis.Redis()` class and
the :class:`redis.StrictRedis()` classes.  It also contains functions to patch the redis module to use these new
extended classes.
